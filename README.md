### Colors

The color scheme can be customized by modifying the CSS variables in `styles.css`:

```css
:root {
  --primary-color: #1a1a1a;
  --accent-color: #ff6b35;
  --text-primary: #ffffff;
  --text-secondary: #b8b8b8;

```
